## How to Read

1. **Latest First:** Use `ls -r consolidations/` to see newest consolidations first
2. **For Current State:** Read `CONSOLIDATION_NOV_03_*.md` (Phase 1-2)
3. **For History:** Work backwards through dates to understand evolution
4. **Naming Pattern:** `CONSOLIDATION_YYYY_MM_DD_PHASE_N.md` sorts chronologically
